# Hadith Tab - Daily Prophetic Wisdom

Hadith Tab is a minimalist browser extension designed to reconnect users with the teachings of Prophet Muhammad (peace be upon him). Every time you open a new tab, you are greeted with a beautiful, hand-picked Hadith set against a calming, modern backdrop.

## Why Hadith Tab?
The goal is to create an immersive spiritual experience. While most New Tab extensions are cluttered, Hadith Tab uses glassmorphism and dynamic gradients to ensure the Hadith remains the focus, encouraging the user to pause, read, and reflect.

## Key Features
*   Curated Wisdom: Access a library of authentic Hadiths focusing on ethics, spirituality, and knowledge.
*   Aesthetic Design: A modern UI featuring frosted-glass effects and smooth background animations.
*   Typography Focus: Utilizes elegant Arabic fonts for maximum readability and visual appeal.
*   Quick Refresh: Explore more wisdom instantly with the Next Hadith interactive button.
*   Lightweight: Zero tracking, zero ads, and lightning-fast performance.

## Built With
*   HTML5 and CSS3: Custom CSS animations and responsive layouts.
*   JavaScript (ES6+): Pure Vanilla JS for randomizing content logic.
*   Manifest V3: Built using the latest Chrome Extension standards for security and longevity.

## Installation (Developer Mode)
1.  Clone or Download this repository to your local machine.
2.  Open Chrome or Edge and navigate to chrome://extensions/.
3.  Enable Developer mode in the top right corner.
4.  Click Load unpacked and select the project folder.
5.  Open a new tab and enjoy.
